# **App Name**: Akein's Ink

## Core Features:

- Post Listing: Display a list of blog posts with titles and short descriptions.
- Post Details: Display the full content of individual blog posts.
- About Me: Show a profile page for Akein Dikith, with biographical information and a photo.
